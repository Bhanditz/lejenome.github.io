Install

Look on INSTALL.md file.

Unstable builds are available but may be not up to date and unusable.

  • windows x86 (VS build)
  • windows x86 (MinGW build)
  • 64_68 DEB
  • 64_68 RPM
  • 64_86 linux (TGZ)
  • 64_68 Mac OS X Bundle
  • Android apk
  • Web version
